App Name: CineVibe – Discover Movies You Love, Hassle-Free


Purpose:
Enable users to explore trending movies.
Provide a simple way to search for specific films.


Key Features:
Dynamic browsing of trending movies.
Quick and efficient search functionality.
Detailed movie rating information.
Sleek, modern, and user-friendly interface.
Fully responsive design that works on any device.


Tech Stack:
React.js: For building the interactive user interface.
Appwrite: For managing backend services such as data storage and authentication.
TailwindCSS: For styling the application with a clean, modern look.
